# Sistema de Gestión Web - Proyecto Final

## Descripción del Proyecto
Repositorio oficial para el desarrollo del módulo final. Este proyecto implementa una arquitectura MVC para la gestión de usuarios y administración de recursos, integrando una interfaz responsiva y segura.

## Stack Tecnológico
* **Backend:** PHP 8.1 / Laravel (Simulado)
* **Frontend:** HTML5, SCSS pre-procesado
* **Control de Versiones:** Git & GitHub
* **Metodología:** Scrum / Agile

## Estado del Proyecto
El proyecto se encuentra en fase de **Code Freeze** (congelamiento de código) previo al despliegue en producción. Se están realizando las últimas auditorías de calidad y resolución de tickets críticos del Backlog.

## Instalación Local
1. Clonar el repositorio:
   `git clone https://github.com/[TU-USUARIO]/[NOMBRE-DEL-REPO].git`
2. Instalar dependencias:
   `composer install`
   `npm install`
3. Ejecutar servidor:
   `php artisan serve`
